    Shortcuts & interaction

    • 1 / 2 / 3 — Switch Wings / Rooms / Graph view
    • R — Reset camera
    • / — Focus search
    • L — Toggle labels
    • Esc — Close this dialog; then clear search; then unpin; then clear selection
    • Search — type a wing or room fragment; Enter frames in Graph · Alt+N / Alt+P next/prev match
    • Click wing (Wings) — Enter that wing’s rooms
    • Click room (Rooms) — Select & center
    • Click node (Graph) — Focus (inspector lock) & frame
    • [ / ] — With an active route: step hops on the highlighted path; otherwise step along connected rooms (graph) · U — Prior focus
    • Route — In Graph, pick a room: set route start, then Route to here (or Route from search). The scene highlights a path on explicit MCP edges only (same rules as tunnel discovery). Disconnected rooms are valid. Stock MemPalace MCP does not support persisting arbitrary room-to-room graph links — see docs/MCP_CONNECTION_CAPABILITIES.md.
    • Click empty canvas — Clear selection (unless pinned); subtle canvas cue
    • Breadcrumb — Navigate: All wings → wing → refocus room
